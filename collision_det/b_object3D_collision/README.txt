In order to run this, do the following:

1. Add the JARs "LPSOLVESolverPack.jar" and "SCPSolver.jar" to your build path.
2. Set the JRE to 1.8 32-bit (specific version doesn't seem to matter)

Run times:

64-bit, no improvements besides fixing recursion-depth bug: 0.513s
32-bit, no improvements besides fixing recursion-depth bug: 0.549s
